Sobre Mim

Olá! Me chamo Eric e sou estudante de Gestão da Tecnologia da Informação na FATEC de Guaratinguetá. Tenho interesse em desenvolvimento de sistemas, análise de dados e tecnologias que otimizam processos e facilitam o dia a dia das empresas. Sou curioso, dedicado e estou sempre em busca de aprender coisas novas, tanto na prática quanto na teoria. Busco oportunidades para aplicar meus conhecimentos em projetos reais, crescer profissionalmente e contribuir com soluções criativas e eficientes.

Meus Projetos

PontoDoce

Durante um trabalho acadêmico na FATEC, meu grupo e eu desenvolvemos um site de vendas para a empresa fictícia PontoDoce, especializada na comercialização de doces artesanais. O projeto foi construído utilizando HTML, CSS e JavaScript para o front-end, com MongoDB no back-end para o armazenamento de dados. O objetivo foi simular uma aplicação real de e-commerce, focando na experiência do usuário e na funcionalidade do sistema.

AlpTêxtil

Desenvolvemos um software personalizado de gerenciamento de estoque para a empresa AlptTêxtil, que atua na venda de tecidos. Utilizamos HTML, CSS e JavaScript no desenvolvimento do sistema, com uma API integrada a um banco de dados MySQL para realizar o controle completo dos produtos em tempo real. O projeto também incluiu dashboards interativos criados no Power BI, conectados diretamente ao banco de dados, permitindo análises e tomadas de decisão rápidas com base em dados atualizados ao vivo.
